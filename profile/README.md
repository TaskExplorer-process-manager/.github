# TaskExplorer — Portable Advanced Task Manager for Mac

## About TaskExplorer

TaskExplorer provides a complete process management environment for users who need detailed information about running tasks. TaskExplorer combines process list, memory usage, open files, network connections, loaded libraries, and code signature verification. TaskExplorer enables users to monitor, analyze, and terminate processes. TaskExplorer runs on desktop platforms. TaskExplorer is trusted by security researchers, system administrators, developers, and power users worldwide.

## Deep Dive — Process Management Notes

Process list: PID, name, user, CPU, memory, parent PID, start time
Memory: real memory, virtual memory, shared memory, resident size
Open files: file descriptors, sockets, pipes, devices
Network: TCP/UDP connections, local/remote addresses, ports, state
Libraries: loaded dynamic libraries (dylibs), frameworks, bundles
